| 环境 | Linux |

安装yeoman，生成express

```
cnpm install -g yo
cnpm install -g generator-express
```

linux
创建项目之前要执行

```
mkdir -p /root/.config/configstorechmod g+rwx /root /root/.config /root/.config/configstorechmod -R 777 /root/.npm
```

之后开始配置项目

```
yo express
```
